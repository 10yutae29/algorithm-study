- %d의 의미 - 문자열 중간에 % d 가 있으면 이부분에 정수타입의 숫자가 올거구나를 표현

- %d 뒤에 있는 것은 %+앞에 넣을 것

- f스트링처럼 %d 사용가능

- %s 는 문자열 출력

- 반복문 안에 들여쓰기를 잘못하여 print문을 넣으니 계속 출력되었다

- 반복문 안의 들여쓰기의 중요성

- in range 범위 헷갈리지 않기

## ◆ SWEA 4831 전기버스 ◆

- range (a, b, c) : a부터 b-1까지 c만큼 건너뛴다

- range(N, 0, -1) -> N부터 1까지 -1씩 증가 (내림차순)

- while과 for문 등 반복문, 조건문의 들여쓰기에 따라 값이 완전히 달라져서 잘 파악해야겠다.
